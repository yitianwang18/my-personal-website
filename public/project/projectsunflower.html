Project Sunflower

Project Sunflower is a browser extension that is intended to help students avoid distraction while learning remotely.

The extension can be configured to include the websites that often causes distractions while learning such as facebook, twitter, youtube, and so on. Once the websites are configured, user can setup an timer to start a study session, and no access to those websites will be granted during the study session.

User would be awarded by the growth of a sunflower on the popup window. The friend system allows user to see friends' progress on the leaderboard and help encourage each other.
